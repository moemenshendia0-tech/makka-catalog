# مكة للأدوات الصحية — كتالوج التشطيب

> **الرابط المباشر للكتالوج:**  
> https://moemenshendia0-tech.github.io/makka-catalog/  
> يعمل فقط إذا كان ملف **`index.html`** موجوداً في جذر المستودع (ليس README فقط).

كتالوج إلكتروني تفاعلي لعرض منتجات التشطيب (أدوات صحية، قواعد، أحواض، بيديه، بانيو) من شركات: **ليسكو**، **كليوباترا**، **بليزا**، **سان بيور**.

يعمل على الموبايل والكمبيوتر، مع بحث، تصفية حسب الشركة والقسم، وتواصل مباشر عبر واتساب.

---

## هيكل المجلد

يجب أن تكون الملفات بهذا الشكل (نفس مستوى ملف HTML):

```
makka-catalog/
├── README.md
├── index.html                    ← الصفحة الرئيسية (انسخ من makka_catalog_updated.html)
├── makka_catalog_updated.html    ← نسخة التطوير (اختياري)
├── ليسكو/
│   └── (صور المنتجات)
├── كليوباترا/
│   └── (صور المنتجات)
├── بليزا/
│   └── (صور المنتجات)
└── سان بيور/
    └── (صور المنتجات)
```

> **مهم:** بدون مجلدات الصور، الصفحة تفتح لكن **الصور لن تظهر**.

---

## التشغيل محلياً

1. ضع ملف `index.html` ومجلدات الصور في مجلد واحد.
2. افتح `index.html` من المتصفح، أو شغّل خادماً بسيطاً:

```bash
# Python 3
python -m http.server 8080
```

ثم افتح: `http://localhost:8080`

## النشر على الإنترنت

### الطريقة 1 — Netlify Drop (الأسهل عند وجود ملفات كثيرة)

1. ادخل إلى [app.netlify.com/drop](https://app.netlify.com/drop)
2. اسحب **المجلد كامل** (HTML + كل مجلدات الصور)
3. انسخ الرابط الذي يظهر وشاركه مع العملاء

### الطريقة 2 — GitHub Pages

**لا ترفع كل الملفات دفعة واحدة من موقع GitHub** (حد ~100 ملف في المرة). استخدم أحد التالي:

- [GitHub Desktop](https://desktop.github.com) — اسحب المجلد كاملاً ثم Push
- أو ارفع على **دفعات**: HTML أولاً، ثم كل مجلد صور لوحده

**تفعيل GitHub Pages:**

1. المستودع → **Settings** → **Pages**
2. Branch: `main` — Folder: `/ (root)`
3. الرابط: `https://اسم-المستخدم.github.io/اسم-المستودع/`

> للصفحة الرئيسية تلقائياً، سمِّ الملف `index.html`.

### الطريقة 3 — Git من الطرفية

```bash
git init
git add .
git commit -m "إضافة كتالوج مكة للأدوات الصحية"
git branch -M main
git remote add origin https://github.com/اسمك/makka-catalog.git
git push -u origin main
```
